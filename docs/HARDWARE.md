# x-Nord OS Hardware Requirements

## Minimum

| Component | Requirement |
|-----------|--------------|
| CPU | 64-bit x86_64 (amd64) |
| RAM | 8 GB |
| Storage | 30 GB |
| Display | 1024x768 or higher |

## Recommended (for AI Assistant)

| Component | Requirement |
|-----------|--------------|
| RAM | 16 GB |
| Storage | 50 GB SSD |
| GPU | Optional - NVIDIA/AMD for acceleration |

## AI Model Requirements

- **Llama 3.2 3B (default):** ~2 GB disk, 8 GB RAM
- **Llama 3.1 8B (optional):** ~5 GB disk, 16 GB RAM

## Supported Hardware

x-Nord OS is based on Ubuntu 24.04 LTS and supports the same hardware as Kubuntu 24.04, including:

- Intel and AMD processors
- NVIDIA, AMD, and Intel graphics
- Common Wi-Fi and Ethernet adapters
